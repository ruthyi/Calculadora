# Calculadora JS
Calculadora con algunas operaciones:  https://ruthyi.github.io/calculadora/

Tecnologías Aplicadas:
* HTML
* Css
* JavaScript

Operaciones:
* Suma
* Resta
* Multiplicacion
* Divisón
* Potenciación
* Radicacion
